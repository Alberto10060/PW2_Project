 Proyecto de programacion web 2

 En este proyecto crearemos una aplicacion web que tenga de tematica la distribucion de noticias y datos variados.

 Info Sphere es una página web dedicada a la creación y difusión de noticias, por medio de publicaciones donde se compartirán diferentes contenidos multimedia, ya sea imagen, video y texto. En esta se puede seguir a más usuarios para ver sus publicaciones u o noticias, de esta manera también se puede interactuar con estos comentando en sus publicaciones.
 Esta página web tendría como inspiración a “Twitter/X” 

 Dentro del repositorio se tendran primeramente 3 carpetas del proyecto:
 
 -La principal de raiz estaran los HTML, CSS y PHP
 
 -la de SQL estaran los scripts para los comandos y creaciond de la base de datos

 -la de Resources estaran todos los materiales usables para la pagina web y el manejo de los usurios
